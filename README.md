# 🗡️ 江湖行 — Python 武侠学习游戏

> **以武悟码，以码证道**

在金庸武侠世界里闯荡江湖——学武功就是学 Python，修炼就是练编程，战斗就是做题，通关之日，就是 Python 出师之时。

## ✨ 特色

- 🎮 **金庸宇宙**：射雕英雄传、天龙八部、笑傲江湖、鹿鼎记、华山论剑五大篇章
- ⚔️ **武功 = 编程**：每一门武功对应一个 Python 知识点（降龙十八掌 = for 循环，独孤九剑 = 函数定义）
- 📜 **70+ 任务**：主线剧情 + 支线任务 + 隐藏彩蛋
- 📝 **76+ 编程题**：从 print() 到设计模式，融入武侠故事
- 🗺️ **17 个地点**：五大区域，NPC 好感度系统
- 🎲 **随机事件**：路遇山贼、发现宝箱、顿悟时刻……

## 🎮 游戏玩法

```
夜雨。破庙。

你从昏迷中醒来，浑身湿透，头痛欲裂。
身边只有一个破包袱，里面一张纸条和一个竹简。

纸条上写着：
    「此子根骨奇佳，望有缘人收留。
      切记——江湖险恶，唯有智谋可保性命。
      此子不懂武功，但天资聪颖，
      若能以"编程之道"参悟武学，必成大器。」

你必须立刻做出选择——
  [1] 躲在佛像后面（学习 print 输出隐身咒语）
  [2] 假装是庙里的小和尚（学习 input 交互）
  [3] 抄起地上的木棍硬刚（学习变量和运算）
  [4] 从后窗逃跑（学习循环）
```

### 武功 ↔ Python 知识对照

| 基础内功 | 进阶武功 | 高阶绝学 |
|---------|---------|---------|
| 吐纳术 → `print()` | 独孤九剑 → `def` 函数 | 乾坤大挪移 → 装饰器 |
| 内息运转 → 变量 | 降龙十八掌 → `class` 面向对象 | 六脉神剑 → 生成器 |
| 临敌判断 → `if/else` | 太极拳 → 递归 | 葵花宝典 → 多线程 |
| 降龙掌 → `for` 循环 | 凌波微步 → `try/except` | 天龙八音 → 正则表达式 |
| 桃花阵 → `list` | 北冥神功 → `import` 模块 | 易筋经 → 数据结构 & 算法 |

## 🚀 快速开始

### 环境要求

- Python 3.12+
- pip

### 安装运行

```bash
# 克隆仓库
git clone https://github.com/your-username/jianghu.git
cd jianghu

# 安装依赖
pip install -r requirements.txt

# 启动游戏（终端版）
python main.py
```

### Web 版（开发中）

```bash
# 启动 Web 服务器
python web/app.py

# 浏览器打开 http://localhost:5000
```

## 🎯 设计理念

1. **剧情驱动学习**：你不是在做题，你是在闯荡江湖
2. **金庸风味**：NPC 性格鲜明，对话有原著味道
3. **趣味细节**：客栈菜单是编程梗（「列表推导炖鸡」），Bug 叫「蛊虫」，死亡画面是「你的程序抛出了未捕获的异常 😵」
4. **循序渐进**：从 `print("阿弥陀佛")` 到实现完整的武林排行榜系统

## 🛠️ 技术栈

- **语言**：Python 3.12+
- **终端 UI**：Rich（彩色面板、表格、进度条）
- **代码沙箱**：subprocess 隔离执行，超时保护
- **数据存储**：JSON 文件（角色存档、题库）
- **Web 版**（开发中）：Flask + WebSocket
